# Adil Munawar's Personal Portfolio

This is the official repository for my personal portfolio website, built to showcase my skills, projects, and professional background as a developer.
[live preview](https://adilmunawar.vercel.app)

The live site is built and deployed with Vercel.

## Features

* **Hero Section:** A modern, engaging introduction.
* **About Me:** A detailed section describing my experience and passion.
* **Skills Section:** A dynamic logo loop displaying the technologies I work with.
* **Projects:** A carousel showcasing my major projects (including Aditron, AdiNox, and AdiZen).
* **Contact Form:** A functional form for visitors to get in touch.
* **Dynamic Background:** Interactive background built with Spline (or other WebGL technologies).

## Tech Stack

This project is built using a modern, component-based tech stack:

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
* **Deployment:** [Vercel](https://vercel.com/)
